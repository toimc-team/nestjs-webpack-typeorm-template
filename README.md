# 项目说明

## 安装

```bash
$ npm install
```

## 运行脚本

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# hot mode -> 但是注意webpack有一些支持不好的地方，比如GraphQL https://docs.nestjs.cn/8/recipes?id=%e7%83%ad%e9%87%8d%e8%bd%bd
$ npm run start:hot

# production mode
$ npm run start:prod
```

## 测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
